# v0.1.0 — Initial release

First release of the **MotoGP** custom integration for Home Assistant — bringing the
MotoGP, Moto2 and Moto3 calendar, standings and results into HA via a native calendar
entity and sensors. Installable through HACS as a custom repository.

## ✨ Features
- **Calendar entity** listing every Grand Prix round of the season.
- **Next-event sensor** (timestamp) with circuit, countdown and full session schedule attributes.
- **Per-class standings sensors** — championship leader with the top 5.
- **Per-class latest-result sensors** — most recent race podium.
- **UI config + options flow** to pick racing classes (MotoGP / Moto2 / Moto3) and the refresh interval.
- `DataUpdateCoordinator` polling the public MotoGP JSON API (no account or API key required).
- Diagnostics support and a test suite backed by recorded API fixtures.

## 📦 Installation
Add `golaz777/HAMotoGP` as a **custom repository** (type: Integration) in HACS, install
**MotoGP**, restart Home Assistant, then add it via **Settings → Devices & Services → Add Integration → MotoGP**.
